Jawaban :

  • Cara Kerja Website Sebuah website terdiri dari dua bagian utama, yaitu sisi client dan sisi server. Sisi Client Sisi client adalah perangkat komputer yang digunakan oleh pengguna untuk mengakses website. Sebuah website dapat diakses melalui berbagai perangkat client, seperti komputer, laptop, smartphone, dan tablet. Ketika pengguna membuka sebuah website, browser yang digunakan oleh pengguna akan mengirimkan permintaan ke server web. Permintaan ini berisi informasi tentang halaman website yang ingin diakses oleh pengguna. Sisi Server Sisi server adalah perangkat komputer yang menyimpan data dan aplikasi website. Server web akan menerima permintaan dari browser pengguna dan mengirimkan kembali halaman website yang diminta oleh pengguna. Halaman website yang dikirimkan oleh server web terdiri dari berbagai file, seperti HTML, CSS, dan JavaScript. File-file tersebut akan ditampilkan oleh browser pengguna dan membentuk tampilan halaman website yang terlihat oleh pengguna.
  • Hubungan antara Web Server, Domain, dan Hosting Web server, domain, dan hosting adalah tiga komponen penting yang dibutuhkan untuk membuat sebuah website. Web server adalah perangkat komputer yang menyimpan data dan aplikasi website. Domain adalah alamat website yang digunakan oleh pengguna untuk mengakses website. Domain biasanya terdiri dari dua bagian, yaitu nama domain dan ekstensi domain. Nama domain adalah nama yang unik untuk sebuah website, sedangkan ekstensi domain menunjukkan jenis organisasi yang mengelola domain tersebut. Hosting adalah layanan yang menyediakan ruang penyimpanan untuk data dan aplikasi website. Hosting biasanya disediakan oleh perusahaan penyedia layanan hosting.
